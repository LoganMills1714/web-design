SQL

SQL stands for structured query language and is a database used by most websites. SQL is often used to store user data such as usernames and passwords. When you input a username for example, it's put into a search query that might look something like SELECT 'user' AND 'pass' FROM users. If you look at the previous statement, instead of inputting user you could input user'--, the apostrophe ends the statement early and the two dashes comment out the ending apostrophe allowing you to bypass. You could also insert operators like the asterisk which will select all data from the database. These types of attacks are very common and can be mitigated by making sure you remove any special characters from inputted data
